Использую для сдачи ДЗ тестовый проект, который в данный момент изучаю.
Использую сборщик webpack и все инструменты из модулей E12,E13.


Для установки зависимостей выполните команду в терминале

npm i

Для запуска сервера в режиме разработки выполните команду в терминале
npm run start:dev

Для запуска JSON-сервера выполните команду в терминале

json-server --watch db.json
